# PacketHero - Roadmap

## Objective

Build PacketHero incrementally.

Each version must be playable and testable.

The priority is to validate gameplay and learning concepts before adding complexity.

---

# MVP 0.1 - First Playable Prototype

## Goal

Create a complete playable loop from start to finish.

The player must be able to:

* Start the game
* Move inside the house
* Interact with devices
* Discover the first network issue
* Unlock Physical Vision
* Solve the problem
* Receive XP

---

## Features

### Main Menu

Buttons:

* New Game
* Continue (disabled)
* Settings

---

### Home Network

Simple 2D environment containing:

* Hero
* Mother's PC
* Router
* Ethernet cable

---

### Character Movement

The player can move freely in 2D.

Keyboard:

* Arrow keys
* WASD

Mobile:

* Touch controls (later)

---

### Context Detection

When the hero approaches an object:

Example:

PC of Mom

[Inspect]

---

### Device Information

Initial state:

PC of Mom

A computer used by your mother.

After unlocking Physical Vision:

PC of Mom

Connection: Ethernet

Status: Connected

---

### First Story Mission

Mother says:

"The Internet has not been working for several days."

Objective:

Investigate the problem.

---

### Router Event

The player discovers the router.

Interaction triggers:

* Electrical shock
* Story sequence
* Unlock Physical Vision

---

### Physical Vision

First unlocked power.

The player can now see:

* Network cables
* Connected ports
* Disconnected cables

---

### First Problem

One cable is:

* Disconnected
  OR
* Connected to the wrong port

The player must identify and fix it.

---

### Mission Completion

After fixing the problem:

* Internet works again
* Mother is happy
* XP reward granted

Reward:

+100 XP

---

### XP Screen

Display:

Level 1

XP: 100 / 500

---

## Success Criteria

The MVP is successful when a player can:

1. Start the game
2. Walk around the house
3. Interact with devices
4. Unlock Physical Vision
5. Fix the cable issue
6. Receive XP

Without any developer intervention.
